Supports And Hinges

Scale 0 uses automatic scaling. End releases are exact in this solver.

Method

Plane frame: 3 DOF per node, Euler-Bernoulli members, small displacement stiffness, static condensation for rotational end releases.
Frame3DD export is a 3D input deck embedded in the X-Y plane. Native Frame3DD input does not directly define member-end releases, so released bottom-member behavior is exact here and noted in the export.
